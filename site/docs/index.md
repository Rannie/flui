### 直接使用 FLUI
```bash
# Add dependency in pubspec.yaml
flui: ^0.7.0 

# Install
$ flutter pub get

# Use FLUI now
import 'package:flui/flui.dart'
```

::: tip 提示
请确保 Flutter 的版本 >= v1.7.8+hotfix.1
:::
